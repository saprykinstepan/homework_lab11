# Домашнее задание к работе 11
## Условие задачи
С одномерным массивом, состоящим из n вводимых с клавиатуры положительных и отрицательных целых чисел выполнить следующие вычисления: вычислить количество элементов массива, значение которых большу 10, находящихся на чётных позициях. 
## 1. Алгоритм и блок-схема
### Алгоритм
# Алгоритм

1. Начало  
2. Объявить константу:  
   - MAX_SIZE = 100 — максимальный размер массива  

3. Объявить переменные:  
   - n — количество элементов в массиве  
   - count = 0 — счетчик элементов, удовлетворяющих условию  
   - arr[MAX_SIZE] — массив для хранения элементов  

4. Ввести количество элементов массива:  
   - Вывести "Введите количество элементов в массиве: "  
   - Считать значение n  

5. Проверить корректность размера массива:  
   - Если n > MAX_SIZE:  
     - Вывести "Ошибка: максимальный размер массива - MAX_SIZE"  
     - Завершить программу с кодом ошибки 1  

6. Ввести элементы массива:  
   - Вывести "Введите n элементов массива:"  
   - Для i от 0 до n-1:  
     - Считать arr[i]  

7. Подсчитать элементы, удовлетворяющие условию:  
   - Для i от 0 до n-1:  
     - Если arr[i] > 10 И i % 2 == 0:  
       - Увеличить count на 1  

8. Вывести результат:  
   - Вывести "Количество элементов больших 10, которые находятся на чётных позициях: count"  

9. Конец

## 2. Реализация программы
    #include <stdio.h>
    #include <locale.h>
    #define MAX_SIZE 100
    
    int main() {
        setlocale(LC_ALL, "");
        int n, count = 0;
        printf("Введите количество элементов в массиве: ");
        scanf("%d", &n);
    
        if (n > MAX_SIZE) {
            printf("Ошибка: максимальный размер массива - %d\n", MAX_SIZE);
            return 1;
        }
    
        int arr[MAX_SIZE];
    
        printf("Введите %d элементов массива:\n", n);
        for (int i = 0; i < n; i++) {
            scanf("%d", &arr[i]);
        }
        for (int i = 0; i < n; i++) {
            if (arr[i] > 10 && i % 2 == 0) {
                count++;
            }
        }
        printf("Количество элементов больших 10, которые находятся на чётных позициях: %d\n", count);
    	return 0;
    }
